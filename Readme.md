# Cybersecurity Lab Setup: VirtualBox, Kali Linux, Metasploitable


## Tools & Downloads
- [Oracle VirtualBox](https://www.virtualbox.org/wiki/Downloads)
- [Kali Linux](https://www.kali.org/get-kali/)
- [Metasploitable (Rapid7)](https://information.rapid7.com/download-metasploitable-2017)

---

## Step 1: Install VirtualBox
- Download installer from Oracle site
- Run setup wizard → accept defaults
- Configure network and storage
- Verify installation by opening VirtualBox Manager

## Step 2: Install Kali Linux
- Download ISO from Kali.org
- Create new VM in VirtualBox → select Linux/Debian
- Allocate RAM & disk space
- Mount ISO → run installation wizard


## Step 3: Install Metasploitable
- Download VM image from Rapid7
- Import appliance into VirtualBox
